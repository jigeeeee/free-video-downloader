# 万能视频下载器 — 需求分析文档

> 版本: v1.2 | 日期: 2026-06-29 | 状态: 下载 + AI 增值 + 任务持久化阶段

## 1. 项目背景

用户下载和处理在线视频时，常见问题包括：

- 平台不提供下载或限制清晰度
- 多平台工具分散，操作复杂
- 下载后仍需要字幕、总结、转文字、提取音频等二次处理
- 批量处理和任务历史缺失
- Bilibili/Douyin 等平台 Cookie 管理麻烦

本项目目标是提供一个本地运行的一站式音视频工具：粘贴链接即可下载，并在本机完成字幕、AI 理解、格式转换和历史管理。

## 2. 目标用户

| 用户类型 | 特征 | 核心需求 |
|---|---|---|
| 学生/学习者 | 保存课程、讲座、教程 | 字幕、总结、问答、笔记 |
| 内容创作者 | 收集素材、剪辑、复盘 | 高清下载、音频提取、格式转换、改写 |
| 普通用户 | 偶尔保存视频 | 简单下载、播放、删除 |
| 高频用户 | 多链接、多平台 | 批量队列、任务历史、失败重试 |

## 3. 已实现核心需求

### 3.1 URL 解析和下载

- 输入视频链接，自动识别平台
- 展示标题、封面、时长、作者、格式
- 选择格式后提交下载任务
- 前端轮询实时进度
- 下载完成后进入视频库

### 3.2 文件库管理

- 列出本地已下载视频
- 支持播放/下载
- 支持删除
- 支持缩略图展示
- 文件访问限制在下载目录，防路径穿越

### 3.3 AI 增值服务

- 字幕提取
- AI 总结
- 思维导图
- AI 问答
- AI 内容改写
- 字幕翻译
- 本地 Whisper 转录

### 3.4 任务中心和历史

- SQLite 持久化任务记录
- 查询所有任务
- 查询单个任务
- 失败任务重试
- 文件历史和 AI 结果历史

### 3.5 格式转换

- 提取音频为 MP3
- 转换封装格式
- 压缩 MP4
- 转换任务进入任务中心

## 4. 非功能性需求

| 需求 | 当前实现 |
|---|---|
| 性能 | 单机并发默认 2，可通过 `MAX_CONCURRENT` 调整 |
| 可用性 | 错误信息进入任务记录；健康检查报告依赖状态 |
| 兼容性 | Chrome/Edge/Safari 现代浏览器；前端响应式 |
| 安全性 | 文件接口防路径穿越；上传文件类型和大小限制 |
| 可维护性 | 下载、Cookie、存储、转换、启动检查拆分模块 |
| 持久化 | SQLite 保存任务、文件、AI 结果 |

## 5. 技术约束

- 后端：Python 3.12+ / FastAPI / yt-dlp
- 前端：Vue 3 + Vite + Tailwind CSS v4
- 存储：文件系统 + SQLite
- 部署：本地单机运行
- 音视频：ffmpeg 必需
- AI：DeepSeek API Key 可选；无 Key 时下载功能不受影响
- 转录：Whisper 依赖本地模型与计算资源
